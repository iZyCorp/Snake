## Snake O:

1. Clone the repository
```shell
git clone https://github.com/iZyCorp/Snake.git
```

2. Initialize a python env
```shell
python3 -m venv .venv
```

3. Activate the env
Windows :
```shell
.venv/Scripts/activate
```

Mac / Linux :
```shell
.venv/bin/activate
```

4. Install the dependencies (in root path)
```shell
pip install .
```
